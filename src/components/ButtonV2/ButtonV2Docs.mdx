# Button 2

A Fancy button with sick hover effect

```js
import { ButtonV2 } from "poshkit";

function App() {
  return (
    <>
      <ButtonV2 label="ButtonName" />;
    </>
  );
}
```

### Props

| Name         | Type                    | Description                                | Required |
| ------------ | ----------------------- | ------------------------------------------ | -------- |
| `color`      | HexColorString          | Changes the text color of the button       | No       |
| `background` | Color or HexColorString | Changes the background color of the button | No       |
| `label`      | string                  | Changes the label of the button            | Yes      |
| `size`       | size                    | Changes the size of the button             | No       |
| `radius`     | radius                  | Changes the border-radius of the button    | No       |
