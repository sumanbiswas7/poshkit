# LoaderV3

A Fancy loader with moving eye animation

```js
import { LoaderV3 } from "poshkit";

function App() {
  return (
    <>
      <LoaderV3
        loading={true}
        size="sm"
        eyeBackground="#00bd6c"
        eyeballColor="#1d1d1d"
      />
    </>
  );
}
```

### Props

| Name            | Type                    | Description                                    | Required |
| --------------- | ----------------------- | ---------------------------------------------- | -------- |
| `size`          | Size                    | Changes the size of the component              | No       |
| `loading`       | Boolean                 | Dispalys the loader if true else displays null | No       |
| `eyeBackground` | Color or HexColorString | Changes background color of the eyeball        | No       |
| `eyeballColor`  | Color or HexColorString | Changes color of the eyeball                   | No       |
